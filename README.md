## Ejecutar la práctica
Comando para ejecutar la simulación (desde el directorio proyectoRobotica-v3.0):
```
./runTestLineSim
```
Si no existen permisos de ejecución, utilizar el comando:
```
chmod a+x runTestLineSim
```
